# common-setting
common setting includes 
.gitconfig - set for git alias
.bashrc 
.bash_profile 
and etc.

## useage

* cd into your home dir :
`cd ~`

* copy .gitconfig file :
`cp -rp .gitconfig ./`


* test :
cd a git directory and input: `git st`
